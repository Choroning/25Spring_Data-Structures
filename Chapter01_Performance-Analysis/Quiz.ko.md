## Q1. 공간 복잡도 정의

**문제:** 프로그램의 공간 복잡도를 정의하라. 총 공간 요구량을 구성하는 두 가지 요소는 무엇인가?

**정답:** 공간 복잡도 $S(P)$는 프로그램 $P$가 필요로 하는 총 메모리양이다. 고정 부분 $c$(명령어 공간, 단순 변수, 상수)와 인스턴스 특성(입력 크기, 재귀 깊이, 동적 메모리)에 따라 달라지는 가변 부분 $S_p$로 구성된다.

**해설:** $S(P) = c + S_p(\text{instance characteristics})$. 고정 부분은 입력에 독립적이며, 가변 부분은 문제 크기에 따라 증가한다.

## Q2. 반복문의 단계 계수

**문제:** 다음 코드 조각의 단계 수를 구하라:
```cpp
for (int i = 0; i < n; i++)
    for (int j = 0; j < n; j++)
        x += 2;
```

**정답:** 총 단계 수는 $2n^2 + 2n + 1$이다.

**해설:** 외부 반복문 검사는 $n+1$회 실행되고, 내부 반복문 검사는 $n(n+1)$회 실행되며, 대입문 `x += 2`는 $n^2$회 실행된다. 합산하면: $(n+1) + n(n+1) + n^2 = 2n^2 + 2n + 1$.

## Q3. 재귀 공간 복잡도

**문제:** 크기 $n$인 배열의 합을 계산하는 재귀 함수의 공간 복잡도는 얼마인가?

**정답:** $S_p = O(n)$.

**해설:** 각 재귀 호출은 반환 주소, 매개변수, 지역 변수를 포함하는 프레임을 호출 스택에 추가한다. 재귀 깊이가 $n$이므로 가변 공간은 $n$에 비례한다.

## Q4. 최선, 최악, 평균 경우

**문제:** 크기 $n$인 정렬되지 않은 배열에 대한 순차 검색의 최선, 최악, 평균 시간 복잡도를 기술하라.

**정답:** 최선 경우: $O(1)$, 최악 경우: $O(n)$, 평균 경우: $O(n)$.

**해설:** 최선 경우는 대상이 첫 번째 위치에 있을 때(비교 1회)이다. 최악 경우는 대상이 마지막 위치에 있거나 존재하지 않을 때(비교 $n$회)이다. 평균 경우는 균등 확률을 가정하면 평균 $n/2$회의 비교가 필요하며, 이는 $O(n)$이다.

## Q5. 성능 측정

**문제:** 경과 시간(wall-clock time)과 CPU 시간의 차이를 설명하라. 알고리즘 성능 비교에 어느 것이 더 적절한가?

**정답:** 경과 시간은 실제 경과된 총 시간이다(I/O 대기, 문맥 교환 포함). CPU 시간은 CPU가 해당 프로세스를 실행하는 데 소비한 시간만을 의미한다. 외부 요인을 배제하므로 알고리즘 비교에는 CPU 시간이 더 적절하다.

**해설:** 경과 시간은 시스템 부하에 따라 크게 변동할 수 있는 반면, CPU 시간은 알고리즘이 수행한 계산 작업을 더 정확하게 반영한다.
