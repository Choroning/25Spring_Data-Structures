## Q1. 이진 트리 성질

**문제:** 이진 트리의 높이가 $h = 5$이다. 최소 및 최대 노드 수는?

**답:** 최소: $h = 5$개 노드 (퇴화/편향 트리). 최대: $2^5 - 1 = 31$개 노드 (포화 이진 트리).

**해설:** 높이 $h$인 이진 트리는 최소 $h$개의 노드(루트에서 리프까지의 단일 체인)와 최대 $2^h - 1$개의 노드(모든 레벨이 완전히 채워짐)를 가진다. $n$개 노드에 대한 최소 높이는 $\lceil \log_2(n+1) \rceil$이다.

## Q2. 순회 순서

**문제:** 다음 이진 트리에 대해 전위, 중위, 후위, 레벨 순서 순회를 작성하라:

```
        A
       / \
      B    C
     / \    \
    D   E    F
   /       / \
  G       H   I
```

**답:**
- 전위 (루트, 왼쪽, 오른쪽): A, B, D, G, E, C, F, H, I
- 중위 (왼쪽, 루트, 오른쪽): G, D, B, E, A, C, H, F, I
- 후위 (왼쪽, 오른쪽, 루트): G, D, E, B, H, I, F, C, A
- 레벨 순서: A, B, C, D, E, F, G, H, I

**해설:** 전위 순회는 루트를 먼저 방문한 후, 왼쪽 서브트리와 오른쪽 서브트리를 재귀적으로 방문한다. 중위 순회는 왼쪽 서브트리, 루트, 오른쪽 서브트리 순으로 방문한다. 후위 순회는 두 서브트리를 루트보다 먼저 방문한다. 레벨 순서 순회는 큐를 사용하여 레벨별로 노드를 처리한다.

## Q3. 완전 이진 트리

**문제:** 완전 이진 트리에 21개의 노드가 있다. (a) 높이는? (b) 리프 노드 수는? (c) 정확히 하나의 자식을 가진 노드 수는?

**답:** (a) 높이 = $\lceil \log_2(21+1) \rceil = \lceil \log_2 22 \rceil = \lceil 4.46 \rceil = 5$. (b) 마지막 내부 노드는 위치 $\lfloor 21/2 \rfloor = 10$이다. 따라서 리프 노드 = $21 - 10 = 11$. (c) $21$은 홀수이므로 모든 내부 노드가 2개의 자식을 가진다. $2 \times 10 = 20 \leq 21$이고 $2 \times 10 + 1 = 21 \leq 21$이므로, 노드 10은 2개의 자식을 가진다. 따라서 정확히 하나의 자식을 가진 노드는 0개이다.

**해설:** $n$개 노드를 가진 완전 이진 트리에서 내부 노드는 위치 1부터 $\lfloor n/2 \rfloor$까지 차지한다. $n$이 홀수이면 모든 내부 노드가 정확히 2개의 자식을 가진다. $n$이 짝수이면 노드 $n/2$만 왼쪽 자식만 가진다.

## Q4. 수식 트리 평가

**문제:** 후위 수식 `3 5 + 2 * 7 -`에 대한 수식 트리를 구성하고 평가하라.

**답:** 수식 트리:
```
        -
       / \
      *   7
     / \
    +   2
   / \
  3   5
```
평가: $(3 + 5) = 8$, $8 \times 2 = 16$, $16 - 7 = 9$. 결과: **9**.

**해설:** 후위 수식으로부터 수식 트리를 구성하려면: 왼쪽에서 오른쪽으로 스캔한다. 각 피연산자에 대해 리프 노드를 생성하여 스택에 push한다. 각 연산자에 대해 두 노드를 pop하고(오른쪽 먼저, 그 다음 왼쪽), 연산자로 새 노드를 생성하여 자식으로 연결한 후 새 노드를 push한다. 결과의 후위 순회는 원래의 후위 수식을 반환한다.

## Q5. 배열 vs. 연결 표현

**문제:** 오른쪽 편향 이진 트리에 4개의 노드(a, b, c, d이며 각 노드는 오른쪽 자식만 가짐)가 있다. 배열 표현에 필요한 배열 슬롯 수는? 연결 표현이 선호되는 이유는?

**답:** 노드 위치는 1, 3, 7, 15이다. 배열은 $2^h = 2^4 = 16$개의 슬롯이 필요하지만, 실제로는 4개만 사용된다. 공간 낭비율은 75%이다. 연결 표현은 낭비 공간 없이 $4 \times \text{sizeof(node)}$만 필요하다.

**해설:** 배열 표현에서 오른쪽 편향 트리는 노드를 위치 $1, 3, 7, 15, \ldots$ (오른쪽 자식 공식 $2x+1$을 따름)에 배치한다. 인덱스의 지수적 증가로 인해 막대한 공간이 낭비된다. 연결 표현은 존재하는 노드만 저장하므로, 비완전 트리에 대해 훨씬 공간 효율적이다.
