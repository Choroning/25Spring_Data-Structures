## 📝 레포지토리 소개

이 레포지토리에는 대학 수준의 자료구조 과목을 위해 작성된 코드가 포함되어 있습니다:

- 강의에서 소개한 코드
- 과제 솔루션 (프로그래밍 과제 및 퀴즈)
- 추가적인 연습 문제

본 과목은 배열, 연결 리스트, 스택, 큐, 스킵 리스트, 해싱, 트리, 힙, 이진 탐색 트리 등 핵심 자료구조를 다루며, 성능 분석과 시간 복잡도에 중점을 둡니다.
