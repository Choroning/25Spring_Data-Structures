## Q1. Big-O 증명

**문제:** $5n^2 + 3n = O(n^2)$임을 증명하라.

**정답:** $c = 6$, $n_0 = 3$을 선택한다. 모든 $n \geq 3$에 대해: $5n^2 + 3n \leq 5n^2 + n^2 = 6n^2$ ($n \geq 3$일 때 $3n \leq n^2$이므로). 따라서 $5n^2 + 3n = O(n^2)$이다.

**해설:** Big-O를 증명하려면 모든 $n \geq n_0$에 대해 $f(n) \leq c \cdot g(n)$을 만족하는 상수 $c$와 $n_0$을 찾아야 한다. $3n$을 $n^2$로 상한을 잡았다($n \geq 3$일 때 유효).

## Q2. 점근적 비교

**문제:** 다음을 성장률의 오름차순으로 나열하라: $n^2$, $2^n$, $n \log n$, $n!$, $\log n$, $n$.

**정답:** $\log n < n < n \log n < n^2 < 2^n < n!$

**해설:** 로그 성장이 가장 느리고, 선형, 선형-로그 순서이다. 다항식 성장($n^2$)은 지수($2^n$)에 의해 지배되며, 지수는 계승($n!$)에 의해 지배된다.

## Q3. 마스터 정리

**문제:** 마스터 정리를 사용하여 $T(n) = 4T(n/2) + n$을 풀어라.

**정답:** $T(n) = \Theta(n^2)$.

**해설:** 여기서 $a = 4$, $b = 2$, $d = 1$이다. $\log_b a = \log_2 4 = 2$를 계산한다. $d = 1 < 2 = \log_b a$이므로, 마스터 정리의 경우 1에 해당한다: $T(n) = \Theta(n^{\log_b a}) = \Theta(n^2)$.

## Q4. 코드 복잡도

**문제:** 다음 코드의 시간 복잡도는 무엇인가?
```cpp
for (int i = 1; i < n; i *= 2)
    for (int j = 0; j < i; j++)
        sum++;
```

**정답:** $O(n)$.

**해설:** 외부 반복문은 $\log n$회 실행된다($i = 1, 2, 4, \ldots, n$). 내부 반복문은 각 $i$ 값에 대해 $i$회 실행된다. 총 연산 횟수는 $1 + 2 + 4 + \cdots + n = 2n - 1 = O(n)$이다(등비급수).

## Q5. Theta 표기법

**문제:** $n^2 + 5n = \Theta(n^2)$이라고 말하는 것이 올바른가? 답을 증명하라.

**정답:** 그렇다, $n^2 + 5n = \Theta(n^2)$이다.

**해설:** $c_1 n^2 \leq n^2 + 5n \leq c_2 n^2$을 만족하는 상수 $c_1, c_2, n_0$이 필요하다. 하한: $n^2 + 5n \geq n^2 = 1 \cdot n^2$ ($c_1 = 1$ 선택). 상한: $n \geq 1$일 때 $n^2 + 5n \leq n^2 + 5n^2 = 6n^2$ ($c_2 = 6$, $n_0 = 1$ 선택).
