## 1. 점근적 표기법

### 1.1 동기

정확한 단계 수 계산은 번거롭고 플랫폼에 의존적이다. 점근적 표기법은 함수의 성장률을 기술하는 **기계 독립적인** 방법을 제공한다.

$n \to \infty$일 때 **실행 시간이 어떻게 확장되는지**에 초점을 맞추며, 상수 인자와 저차항은 무시한다.

### 1.2 표기법 개요

| 표기법 | 의미 | 비유 |
|--------|------|------|
| $O(g(n))$ | 상한 | $\leq$ |
| $\Omega(g(n))$ | 하한 | $\geq$ |
| $\Theta(g(n))$ | 엄밀한 한계 | $=$ |
| $o(g(n))$ | 엄격한 상한 | $<$ |
| $\omega(g(n))$ | 엄격한 하한 | $>$ |

## 2. Big-O 표기법

### 2.1 정의

양의 상수 $c$와 $n_0$이 존재하여 다음을 만족하면 $f(n) = O(g(n))$이다:

$$f(n) \leq c \cdot g(n) \quad \text{for all } n \geq n_0$$

Big-O는 성장률에 대한 **점근적 상한**을 제공한다.

### 2.2 예시

- $3n + 2 = O(n)$: $n \geq 2$일 때 $3n + 2 \leq 4n$ ($c = 4$, $n_0 = 2$ 선택)
- $10n^2 + 4n + 2 = O(n^2)$: $n \geq 5$일 때 $10n^2 + 4n + 2 \leq 11n^2$
- $6 \cdot 2^n + n^2 = O(2^n)$: 충분히 큰 $n$에 대해 $6 \cdot 2^n + n^2 \leq 7 \cdot 2^n$

### 2.3 성질

1. **추이성**: $f = O(g)$이고 $g = O(h)$이면, $f = O(h)$
2. **합의 규칙**: $O(f + g) = O(\max(f, g))$
3. **곱의 규칙**: $O(f \cdot g) = O(f) \cdot O(g)$
4. **상수 인자**: 임의의 상수 $c > 0$에 대해 $O(c \cdot f) = O(f)$

> **핵심 포인트:** Big-O는 *최악 경우*의 상한을 기술한다. 함수가 $g(n)$보다 *빠르게 증가하지 않음*을 의미한다.

## 3. Big-Omega 표기법

### 3.1 정의

양의 상수 $c$와 $n_0$이 존재하여 다음을 만족하면 $f(n) = \Omega(g(n))$이다:

$$f(n) \geq c \cdot g(n) \quad \text{for all } n \geq n_0$$

### 3.2 예시

- $3n + 2 = \Omega(n)$: $n \geq 1$일 때 $3n + 2 \geq 3n$
- $n^2 - 5n = \Omega(n^2)$: $n \geq 10$일 때 $n^2 - 5n \geq \frac{1}{2}n^2$

> **핵심 포인트:** Big-Omega는 *하한*을 제공한다. 함수가 *최소한 $g(n)$만큼 빠르게* 증가한다는 의미이다.

---

## 4. Theta 표기법

### 4.1 정의

양의 상수 $c_1$, $c_2$, $n_0$이 존재하여 다음을 만족하면 $f(n) = \Theta(g(n))$이다:

$$c_1 \cdot g(n) \leq f(n) \leq c_2 \cdot g(n) \quad \text{for all } n \geq n_0$$

동치 조건으로, $f(n) = O(g(n))$ **이고** $f(n) = \Omega(g(n))$이면 $f(n) = \Theta(g(n))$이다.

### 4.2 예시

- $3n + 2 = \Theta(n)$: $n \geq 2$일 때 $3n \leq 3n+2 \leq 4n$
- $n^2 / 2 - 3n = \Theta(n^2)$

## 5. Little-o와 Little-omega

### 5.1 Little-o

$f(n) = o(g(n))$은 $\lim_{n \to \infty} \frac{f(n)}{g(n)} = 0$을 의미한다.

예시: $n = o(n^2)$, $\log n = o(n)$

### 5.2 Little-omega

$f(n) = \omega(g(n))$은 $\lim_{n \to \infty} \frac{f(n)}{g(n)} = \infty$를 의미한다.

예시: $n^2 = \omega(n)$

## 6. 일반적인 성장률

### 6.1 성장률 계층

$$O(1) < O(\log n) < O(\sqrt{n}) < O(n) < O(n \log n) < O(n^2) < O(n^3) < O(2^n) < O(n!)$$

### 6.2 실질적 영향

| 복잡도 | $n = 10$ | $n = 100$ | $n = 1000$ | $n = 10^6$ |
|--------|----------|-----------|------------|------------|
| $O(1)$ | 1 | 1 | 1 | 1 |
| $O(\log n)$ | 3.3 | 6.6 | 10 | 20 |
| $O(n)$ | 10 | 100 | 1000 | $10^6$ |
| $O(n \log n)$ | 33 | 664 | 10000 | $2 \times 10^7$ |
| $O(n^2)$ | 100 | 10000 | $10^6$ | $10^{12}$ |
| $O(2^n)$ | 1024 | $10^{30}$ | - | - |

### 6.3 코드에서 복잡도 식별

```cpp
// O(1) - 반복문 없음, 직접 계산
int getFirst(int arr[]) { return arr[0]; }

// O(log n) - 매 반복마다 문제를 절반으로 축소
while (n > 1) { n /= 2; count++; }

// O(n) - n에 비례하는 단일 반복문
for (int i = 0; i < n; i++) { /* ... */ }

// O(n log n) - 로그적 내부 작업을 가진 반복문
for (int i = 0; i < n; i++)
    for (int j = 1; j < n; j *= 2) { /* ... */ }

// O(n^2) - n에 비례하는 중첩 반복문
for (int i = 0; i < n; i++)
    for (int j = 0; j < n; j++) { /* ... */ }
```

## 7. 복잡도 분석 기법

### 7.1 점화식

많은 재귀 알고리즘의 시간 복잡도는 점화식으로 기술된다:

$$T(n) = aT(n/b) + f(n)$$

**마스터 정리(Master Theorem)**: $T(n) = aT(n/b) + \Theta(n^d)$가 주어질 때:

1. $d < \log_b a$이면: $T(n) = \Theta(n^{\log_b a})$
2. $d = \log_b a$이면: $T(n) = \Theta(n^d \log n)$
3. $d > \log_b a$이면: $T(n) = \Theta(n^d)$

### 7.2 예시

**병합 정렬(Merge Sort)**: $T(n) = 2T(n/2) + \Theta(n)$
- $a = 2$, $b = 2$, $d = 1$
- $\log_2 2 = 1 = d$ (경우 2)
- $T(n) = \Theta(n \log n)$

**이진 검색(Binary Search)**: $T(n) = T(n/2) + \Theta(1)$
- $a = 1$, $b = 2$, $d = 0$
- $\log_2 1 = 0 = d$ (경우 2)
- $T(n) = \Theta(\log n)$

### 7.3 분할 상환 분석

일부 연산은 가끔 비용이 크지만 평균적으로는 저렴하다. **분할 상환 분석(Amortized Analysis)** 은 연산 수열에 대한 연산당 평균 비용을 제공한다.

예시: 동적 배열 `push_back`:
- 일반적으로 $O(1)$ (단순 추가)
- 가끔 $O(n)$ (크기 조정 시)
- 분할 상환 비용: 연산당 $O(1)$

## 요약

| 개념 | 핵심 포인트 |
|------|-------------|
| Big-O | 상한: 큰 $n$에 대해 $f(n) \leq c \cdot g(n)$ |
| Big-Omega | 하한: 큰 $n$에 대해 $f(n) \geq c \cdot g(n)$ |
| Theta | 엄밀한 한계: $O$와 $\Omega$ 모두 만족 |
| 성장률 계층 | $1 < \log n < n < n\log n < n^2 < 2^n < n!$ |
| 마스터 정리 | $T(n) = aT(n/b) + \Theta(n^d)$ 점화식을 해결 |
| 분할 상환 분석 | 연산 수열에 대한 평균 비용 |

## 부록

### Big-O의 형식적 증명

**주장**: $2n^2 + 3n + 1 = O(n^2)$

**증명**: $c = 6$, $n_0 = 1$을 선택한다.
$n \geq 1$일 때: $2n^2 + 3n + 1 \leq 2n^2 + 3n^2 + n^2 = 6n^2$. QED.
