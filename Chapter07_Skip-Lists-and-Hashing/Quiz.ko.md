## Q1. 스킵 리스트 탐색 경로

**문제:** 레벨 0-2를 가진 스킵 리스트에 원소 {20, 24, 30, 40, 60, 75, 80}이 있고, 레벨 2에는 {40, 80}, 레벨 1에는 {24, 40, 60, 80}, 레벨 0에는 모든 원소가 있을 때, 키 75의 탐색 경로를 추적하라.

**답:** 레벨 2 헤드에서 시작: 40과 비교(오른쪽 이동), 80과 비교(80 > 75, 아래로 이동). 레벨 1에서: 60과 비교(오른쪽 이동), 80과 비교(80 > 75, 아래로 이동). 레벨 0에서: 75와 비교(발견). 총 비교 횟수: 5.

**해설:** 스킵 리스트 탐색은 최상위 레벨에서 시작하여 다음 키가 대상보다 작은 동안 오른쪽으로 이동한다. 다음 키가 대상을 초과하거나 꼬리인 경우 한 레벨 아래로 내려간다. 이를 통해 상위 레벨에서 원소를 건너뛰어 $O(\log n)$의 기대 탐색 시간을 달성한다.

## Q2. 스킵 리스트 레벨 할당

**문제:** $p = 1/2$인 스킵 리스트에서, 새로 삽입된 원소가 레벨 3으로 지정될 확률은? $n = 16$개 원소에 대해 기대되는 레벨 수는?

**답:** 레벨 3이 될 확률은 $(1/2)^3 = 1/8$이다. $n = 16$일 때, 기대 레벨 수는 $\lfloor \log_2 16 \rfloor + 1 = 5$이다.

**해설:** 각 원소는 확률 $p = 1/2$로 다음 레벨로 승격된다. 레벨 $i$에 도달할 확률은 $p^i = (1/2)^i$이다. 최대 레벨은 $\lfloor \log_{1/p} n \rfloor + 1$이며, $p = 1/2$일 때 $\log_2 n + 1$이 된다.

## Q3. 해시 테이블 선형 탐사

**문제:** 크기 11인 해시 테이블에 해시 함수 $h(k) = k \mod 11$과 선형 탐사를 사용하여 키 20, 31, 24, 9, 35, 42를 삽입하라. 테이블의 최종 상태를 보여라.

**답:**
- $h(20) = 9$ -> 슬롯 9
- $h(31) = 9$ -> 충돌, 슬롯 10으로 탐사
- $h(24) = 2$ -> 슬롯 2
- $h(9) = 9$ -> 9, 10에서 충돌; 슬롯 0으로 탐사
- $h(35) = 2$ -> 2에서 충돌; 슬롯 3으로 탐사
- $h(42) = 9$ -> 9, 10, 0에서 충돌; 슬롯 1으로 탐사

최종 테이블: [0]=9, [1]=42, [2]=24, [3]=35, [4..8]=비어있음, [9]=20, [10]=31.

**해설:** 선형 탐사는 $h_i(k) = (h(k) + i) \mod m$ ($i = 0, 1, 2, \ldots$)으로 충돌을 해결한다. 이는 점유된 셀의 클러스터(1차 클러스터링)를 생성하여, 이후 삽입의 평균 탐사 길이를 증가시킨다.

## Q4. 체이닝 vs. 개방 주소법

**문제:** 해시 테이블에 $n = 100$개의 원소와 $m = 200$개의 슬롯이 있다(적재율 $\alpha = 0.5$). 선형 탐사와 체이닝에서 성공적 탐색의 기대 탐사 횟수를 비교하라.

**답:** 선형 탐사: $\frac{1}{2}(1 + \frac{1}{1-\alpha}) = \frac{1}{2}(1 + \frac{1}{0.5}) = \frac{1}{2}(1 + 2) = 1.5$회. 체이닝: $1 + \frac{\alpha}{2} = 1 + 0.25 = 1.25$회.

**해설:** 체이닝이 1차 클러스터링을 겪지 않으므로 선형 탐사보다 약간 우수하다. $\alpha$가 증가하면 성능 차이가 상당히 커진다. $\alpha = 0.9$일 때, 선형 탐사는 5.5회의 탐사가 필요하지만 체이닝은 1.45회만 필요하다.

## Q5. 스킵 리스트 평균적 경우 분석

**문제:** 역방향 분석을 사용하여 스킵 리스트의 기대 탐색 시간을 유도하라. $C(i)$를 $i$개 레벨을 올라가는 기대 비용이라 하자. 점화식을 세우고 $p = 1/2$에 대해 풀어라.

**답:** 역방향 순회 중 각 노드에서: 확률 $1-p$로 노드에 상위 포인터가 없음(비용 $1 + C(i)$); 확률 $p$로 노드에 상위 포인터가 있음(비용 $1 + C(i-1)$). 점화식: $C(i) = (1-p)(1+C(i)) + p(1+C(i-1))$. $C(0) = 0$으로 풀면 $C(i) = i/p$. $p = 1/2$일 때: $C(\log_2 n) = 2\log_2 n = O(\log n)$.

**해설:** 역방향 분석은 탐색 경로를 역방향(위쪽과 왼쪽)으로 추적한다. 점화식은 $i$개 레벨을 올라가는 데 필요한 기대 단계 수를 포착한다. 각 레벨당 $1/p$의 기대 단계가 소요되고 $\log_{1/p} n$개의 레벨이 있으므로, 총 기대 시간은 $\frac{\log n}{p} = O(\log n)$이다.
